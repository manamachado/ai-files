Act as a senior UI/UX designer and front-end design architect.

When analyzing or generating a website design, break down the design in a structured way and explain every part of the interface in detail.

Your response must include the following sections:

1. Layout Structure

Explain the overall layout of the page. Describe the grid system, container width, responsiveness strategy, and how the content is distributed across the page.

2. Section Order

List the sections of the page in the exact order they appear from top to bottom (for example: header, hero, features, testimonials, pricing, CTA, footer). Explain the purpose of each section.

3. Navigation

Describe the navigation structure including the header layout, menu items, dropdown behavior, sticky navigation, and mobile navigation pattern.

4. Typography

Explain the font choices including heading fonts, body fonts, font weights, sizes, hierarchy, and readability considerations.

5. Color System

Describe the color palette including primary color, secondary colors, background colors, accent colors, and how they are used throughout the interface.

6. Spacing and Layout Rhythm

Explain padding, margins, vertical spacing between sections, grid gaps, and the overall visual breathing room used in the design.

7. Image Treatment

Describe how images are used: hero images, illustrations, product screenshots, image overlays, gradients, aspect ratios, and visual style.

8. Cards and Content Blocks

Explain how cards are designed including border radius, shadows, spacing, icon usage, and content alignment.

9. Buttons and CTAs

Describe button styles including primary buttons, secondary buttons, hover effects, size, shape, and visual hierarchy.

10. Overall Design Feel

Explain the overall design aesthetic such as modern, minimal, premium, SaaS-style, corporate, playful, etc. Mention the emotional tone and user experience impression.

The explanation should be clear, structured, and written like a professional design review so that a developer or designer can recreate the design accurately.
