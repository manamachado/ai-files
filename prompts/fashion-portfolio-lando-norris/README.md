Esse arquivo contém um prompt completo para criação de websites, com instruções detalhadas para um agente de IA gerar landing pages premium com HTML/CSS/JS puro:

Fashion Portfolio (linhas 1–377)

Instrui a criação de uma landing page de portfólio de moda editorial, inspirada no site do Lando Norris (piloto de F1). Os destaques são:

Efeito principal: imagem em preto e branco no hero que revela cores ao passar o mouse, como um pincel pintando cor na foto (brush reveal via clip-path / mask-image).

Estrutura: Hero → About Me → Fashion Skills → Galeria/Slider → Texto circular → Depoimentos → Footer.

Estilo: editorial de moda, minimalista luxuoso, alto contraste P&B com acentos rosa/bege apenas nos elementos revelados.

Inclui instruções refinadas de CSS e JS para o efeito de reveal (linhas 189–366).
